# Dotiam 5

## Zadání 5.1

Přepracuj projekt na hru kde nebude mapa jako ascii,
tedy definice jednotlivých pozic, ale bude definovaná jako graf.
Tedy bude graf s vlastnostmi.

Graf bude obsahovat informace o pozici:
- kodové označení
- atributy pozice
- seznam pozic na které je možné přejít včetně podmínky, 
  kdy je možné přejít: tedy seznam vlastností a vecí které musí mít hráč
  nebo hráč u sebe

## Zadání 5.2

- předělej world.yaml předělej do angličtiny
- zakomponuj příkazy:
  - h, help - vypíše seznam příkazů
  - l, look - popíše scénu
  - g, go - jdi některým z možných směrů
  - x, explore - prozkoumej scénu nebo konkrétní věc
  - c, combine - kombinuj věci
  - p, pickup - seber věc
  - d, drop - polož věc
  - i, inventory - zobraz inventář
  - u, use - použij věc
- rozšiř world.yaml tak, aby bylo možné vyzkoušet všechny příkazy
